# Found For AI Analytics

Privacy-first web analytics. Lightweight, cookie-free alternative to Google Analytics.

This is an **independent, rebranded fork** of [Plausible Analytics](https://github.com/plausible/analytics), maintained by Found For AI.

## Features

- Privacy-focused (no cookies, no personal data collection)
- Lightweight tracking script (< 1KB)
- Real-time dashboard
- Self-hosted or cloud
- GDPR, CCPA, and PECR compliant
- Easy event tracking

## Getting Started

### Self-hosted (Free)

You can self-host this yourself using Docker. See the original Plausible documentation for setup instructions:

- [Self-hosting Guide](https://plausible.io/docs/self-hosting)

### Hosted Version (Recommended)

We offer a managed hosted version at **$49/month** with automatic updates, backups, and support.
